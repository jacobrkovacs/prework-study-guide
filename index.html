HTML

  • The "head" element contains information about the webpage.
  • The "body" element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • I can style HTML elements using the elements as selectors or by using the class attribute as a selector.

Git

  • git status: checks the current branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add the changed files to the staging area of Git
  • git commit -m "Your Message Here": commit to repository
  • git pull origin main: make sure current branch is up to date before opening a PR
  • git push origin branch-name: push up changes to GitHub
  • git checkout main: update local main branch
  • git pull origin main: pull the remote main branch into the local main branch

JavaScript

  • A Variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executed code in a script.
  • Functions need to be called upon in order to be executed.
